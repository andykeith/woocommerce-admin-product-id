Display the product ID in the products list in the admin.

== Description ==

A simple plugin to add the product ID as an extra column to the WooCommerce products list in the WordPress admin.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Go the the Products menu and the ID should be shown as an additional column.
